# Pertemuan-13---PBO-B

 Project ImageViewer
Link Repo :



    Pada project kali ini saya membuat suatu sistem penglihat gambar yang dapat digunakan dengan memasukkan kredensial dengan melakukan login terlebih dahulu. Untuk mencapai fungsi tersebut saya menggunakan 3 class yaitu MainApp yang berperan sebagai class utama (main) selanjutnya loginsystem untuk fitur fitur login yang diperlukan oleh sistem dan selanjutnya imageviewer itu sendiri

Untuk menjalankannya kita dapat memulai melalui class MainApp



    Akan tampil tab seperti itu sebagai landing page untuk kita mengakses imageviewer, apabila kita klik pada tombol mulai aplikasi akan menampilkan tampilan login akun. Disini juga kita dapat mendaftarkan akun apabila sebelumnya belum pernah didaftarkan
 





    Apabila kita berhasil login, maka kita dapat mengakses imageviewer dan menampilkan suatu gambar melalui tampilan tersebut

    


    Apabila login gagal akan tertampil


Sehingga kita dapat mendaftarkan akun terlebih dahulu untuk selanjutnya dapat login dan mengakses fitur fitur yang ada pada imageviewer
